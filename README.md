# Órbita - Manual do Usuário

List all posts from the Órbita, the Brazilian hacker news maintained by the [Manual do Usuário](https://manualdousuario.net/) by Rodrigo Ghedin.

This project is a personal project and is not affiliated with the Manual do Usuário. All content and rights reserved for the Manual do Usuário.

## How to use

1. Install Raycast
2. Install the [Órbita extension](https://www.raycast.com/tcelestino/orbita)
3. Run the `orbita` command
4. Follow the posts in the Órbita
