# Órbita - Manual do Usuário

List all posts from the Órbita, the Brazilian hacker news maintained by the [Manual do Usuário](https://manualdousuario.net/) by Rodrigo Ghedin.

> **Note:** This project is a personal project and is not affiliated with the [Manual do Usuário](https://manualdousuario.net/). All content and rights are reserved for the [Manual do Usuário](https://manualdousuario.net/).

## How to use

1. Install [Raycast](https://www.raycast.com/)
2. Install the [Órbita](https://www.raycast.com/tcelestino/orbita) extension
3. Run the `orbita` command
4. Follow the posts in the Órbita
